1
Save the HTML File
Save this code to your computer as an HTML file
1
Press Win + S → type Notepad → Open it
2
Copy the entire HTML code and paste it into Notepad
3
Click File → Save As
4
Name the file:
music_search.html
5
Set "Save as type" to All Files (*.*)
6
Save to your Desktop → Click Save
⚠️ Make sure the file ends in .html — not .html.txt
2
Install VS Code
Download and install Visual Studio Code — it's free
1
Go to https://code.visualstudio.com
2
Click "Download for Windows"
3
Open the downloaded .exe file
4
Click Next → Next → Install → Finish
💡 VS Code is completely free and made by Microsoft — it is the most popular code editor in the world.
3
Install Live Server Extension
This runs the app at http://127.0.0.1:5500 so Chrome works perfectly
1
Open VS Code
2
Click the Extensions icon on the left sidebar (looks like 4 squares)
3
Search for:
Live Server
4
Install the one by Ritwick Dey → Click Install
5
Press Ctrl + Shift + P → type Open User Settings JSON → select it
6
Add this inside the curly braces:
"liveServer.settings.CustomBrowser": "chrome"
7
Press Ctrl + S to save
✅ No API key needed at all — this app works 100% without any Google configuration!
4
Launch in Chrome
Open the file with Live Server so Chrome loads it correctly
1
Open VS Code
2
Click File → Open File → Find your music_search.html → Click Open
3
Look at the bottom right corner of VS Code → Click "Go Live"
4
Chrome opens automatically at:
http://127.0.0.1:5500/music_search.html
5
In Chrome → click the 🔒 lock icon → Site Settings → Set Pop-ups to Allow
6
Bookmark it — click the ⭐ star in the address bar → name it Music Search App
💡 Every time you want to use the app — open VS Code, click Go Live, then click your Chrome bookmark.
5
How to Use the App
You are ready! No API key needed — here is how everything works
1
Type any artist, song or album in the search bar — e.g. Taylor Swift
2
Select a filter — All / Artist / Song / Album — to narrow results
3
Press Enter or click 🔍 Search — both panels load at the same time
4
Left panel — 12 music platforms (Amazon Music, Spotify, Apple Music etc.) — click any card to open
5
Right panel (Google) has 3 views — use the tabs at the top:
• 🔗 Quick Links — direct Google search links by category (no API needed)
• 🖥️ Live View — Google search embedded live in the panel
• 🌐 Open Google — opens full Google search in a new tab
6
Click any result card to open the page in a new tab
🎉 You are fully set up! Zero API keys needed. Click Close Guide and start searching!
