# 🎵 Music Search Extractor

A high-performance, single-page web application designed to streamline music discovery. This tool extracts search results across **12+ music platforms** and **Google Search** simultaneously, without requiring any complex API configurations or backend setup.

---

## 🚀 Features

* **Multi-Platform Extraction**: Instantly generates search links for Amazon Music, Spotify, Apple Music, YouTube Music, Tidal, SoundCloud, and more.
* **Intelligent Filtering**: Context-aware search modes for **Artists**, **Songs**, and **Albums** to refine results.
* **Dual-Panel Interface**:
    * **Left Panel**: Direct access to dedicated music streaming and database platforms.
    * **Right Panel**: A specialized Google Search suite featuring "Quick Links" for biographies, lyrics, and tour dates.
* **Zero-API Architecture**: Works entirely through dynamic URL construction—no API keys or rate limits required.
* **Modern UI/UX**: Built with a dark-themed, responsive interface featuring CSS variables, smooth transitions, and a built-in step-by-step Setup Guide.

---

## 🛠️ Tech Stack

* **Frontend**: HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+).
* **Design**: Dark mode UI, custom SVG-inspired icons, and responsive layouts.
* **Development Environment**: Optimized for **VS Code** with the **Live Server** extension.

---

## 📖 How to Use

1.  **Clone the Repository**:
    ```bash
    git clone [https://github.com/Sudharsan-B/Music-Search-Extractor.git](https://github.com/Sudharsan-B/Music-Search-Extractor.git)
    ```
2.  **Open in VS Code**: Open the folder and locate `Music_Extractor.html`.
3.  **Run Live Server**:
    * Right-click `Music_Extractor.html` and select **"Open with Live Server"**.
    * Alternatively, click the **"Go Live"** button in the bottom-right corner of VS Code.
4.  **Search**: Type an artist (e.g., *Taylor Swift*) or song name and hit Enter to populate both panels.

---

## ⚙️ Setup Tips

* **Pop-ups**: Ensure your browser allows pop-ups for `127.0.0.1` so that result cards can open in new tabs.
* **Live View**: The "Live View" tab utilizes an iframe for Google Search; note that some browser security settings or Google's headers may restrict embedding in certain environments.
